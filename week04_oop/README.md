# Lab 4: Introduction to Classes and Objects

## IST1012 — Computer Programming II | Difficulty: Easy

---

## Objective

This lab introduces the fundamental building blocks of Object-Oriented
Programming (OOP) in Python: classes, objects, attributes, and methods.
All examples use cybersecurity-themed scenarios to keep the material
relevant to your degree.

## Prerequisites

- Python 3.5 or later installed
- Jupyter Notebook or JupyterLab
- Basic Python knowledge: variables, functions, dictionaries, loops

## What Students Will Learn

| # | Skill |
|---|-------|
| 1 | Define a class with `__init__` |
| 2 | Create and use object instances |
| 3 | Write methods that operate on object data |
| 4 | Build simple cybersecurity utilities using OOP |

## Lab Structure

| Section | Time | Description |
|---------|------|-------------|
| Warm-Up | 5-10 min | Recall basic Python concepts |
| Core Concepts | 10-15 min | Classes, `__init__`, methods |
| Guided Exercises | 25-30 min | 4 hands-on tasks (increasing difficulty) |
| Challenge | 10 min | Build an intrusion log manager |

## How to Run the Notebook

1. Clone or download this repository.
2. Open a terminal in the lab folder.
3. Run `jupyter notebook lab.ipynb`
4. Work through each cell from top to bottom.
5. Write your code in the cells marked `# YOUR CODE HERE`.

## Submission Guidelines

- Complete all exercises in the notebook.
- Ensure every cell runs without errors (Kernel → Restart & Run All).
- Save the notebook and submit the `.ipynb` file via the course portal
  before the posted deadline.
